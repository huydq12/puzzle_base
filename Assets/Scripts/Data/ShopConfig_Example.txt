SHOP CONFIGURATION GUIDE
========================

This guide shows how to configure your shop packs in Unity.

1. CREATE SHOP CATALOG ASSET
   - Right-click in Project window
   - Create > Game > Shop Catalog
   - Name it "ShopCatalog"

2. CONFIGURE CATEGORIES

3. SETUP IN UNITY SCENE
   - Add UIShop prefab to your scene
   - Assign ShopCatalog asset to UIShop component
   - Create tab button prefab with Button + TextMeshProUGUI
   - Create shop item prefab with ShopItem component
   - Assign all UI references in UIShop inspector

4. CURRENCY TYPES
   - Coin: Purchase with in-game coins (playerCash)
   - Diamond: Purchase with premium currency (playerDiamond)
   - IAP: Purchase with real money through app store

5. PURCHASE LIMITS
   - None: Can purchase unlimited times
   - Daily: Can purchase once per day
   - Weekly: Can purchase once per week
   - OneTime: Can only purchase once ever

6. SPECIAL OFFERS
   - Set isSpecialOffer = true
   - Set discountPercent (e.g., 20 for 20% off)
   - Will display special badge in UI
